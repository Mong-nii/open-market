# 🛒 HODU 오픈마켓

바닐라 JavaScript 반응형 오픈마켓 웹사이트

## 🛠️ 기술 스택
HTML5, CSS3, Vanilla JavaScript

## ✅ 주요 기능
- 회원가입/로그인 (실시간 유효성 검사)
- 상품 목록/상세 조회
- 상품 검색
- 반응형 디자인

## 📁 구조
```
├── css/           # 스타일
├── js/            # 기능
├── index.html     # 메인
├── join.html      # 회원가입
├── login.html     # 로그인
└── detail.html    # 상품상세
```

1. 많은부분에서의 LLM의 도움
